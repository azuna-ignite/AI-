ツール紹介

海外マーケティングの実務でよく使うツール・リソースをまとめています。各カードから学んだことを読めます。

SEO・キーワード調査

SEO 有料

Semrush

キーワードリサーチ・競合分析・バックリンク調査・サイト監査が1つで完結する SEO ツール。英語圏の競合サイト調査に強く、海外向けコンテンツ戦略の立案に活用。

学んだことを読む →
SEO 有料

SurferSEO

コンテンツを書きながらリアルタイムで SEO スコアを計測できるライティング支援ツール。上位表示ページとの比較をもとに、見出し構成・語数・キーワード密度を最適化できる。

学んだことを読む →
SEO 無料

Google サーチコンソール

Google が無料提供する検索パフォーマンス分析ツール。クリック数・表示回数・検索クエリ・平均掲載順位を確認でき、インデックス状況のモニタリングにも必須。

学んだことを読む →
キーワード 無料

ラッコキーワード

日本語キーワードのサジェスト・関連語・共起語を一括取得できるツール。日本語コンテンツの企画段階で使うと、ユーザーの検索意図を素早く把握できる。

学んだことを読む →

AI・リサーチ

AI 有料

Claude / Claude Code

Anthropic 製 AI アシスタント。コンテンツの構成作成・リサーチ・文章の推敲に活用。Claude Code は CLI から直接コード編集・ファイル操作が行える開発者向けツール。

学んだことを読む →
AI 無料プランあり

NotebookLM

Google 製の AI リサーチアシスタント。PDFや記事などのソースをアップロードすると、そのドキュメントに基づいた回答・要約・Q&A を生成できる。情報収集と整理に活用。

学んだことを読む →
AI 有料

isgen

AI を活用したコンテンツ生成・マーケティング支援ツール。記事の企画からドラフト作成まで効率化できる。海外マーケティングの量産フローに組み込んで活用。

学んだことを読む →

制作・デザイン

デザイン 有料

Figma

UI/UX デザインのスタンダードツール。LP・バナー・Webサイトのワイヤーフレームからビジュアルデザインまで対応。チームとのリアルタイム共同編集が可能。

学んだことを読む →
デザイン 無料プランあり

Canva

テンプレートが豊富なノーコードデザインツール。SNS バナー・LP ヘッダー・インフォグラフィックの作成に最適。Figma より手軽に使えるため、スピード重視の素材制作に活用。

学んだことを読む →
CMS 無料プランあり

WordPress

世界シェア No.1 の CMS。海外クライアントのサイト制作・ブログ運用に幅広く使われる。プラグインが豊富で SEO 対策（Yoast SEO 等）も拡張しやすい。

学んだことを読む →

情報管理・開発

ノート 無料プランあり

Obsidian

ローカル保存型のマークダウンノートアプリ。学んだ知識をリンクでつなぎ、ナレッジグラフとして管理できる。このナレッジベースサイトのコンテンツ原稿管理にも活用。

学んだことを読む →
開発 無料

GitHub

ソースコードのバージョン管理・公開プラットフォーム。このサイト自体も GitHub Pages でホスティング。学習コンテンツの変更履歴を残しながら管理できる。

学んだことを読む →
