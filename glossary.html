用語集

ライティング・SEO・コンテンツライティング・コピーライティング・セールスライティングの用語をまとめています。

E-E-A-T
Experience / Expertise / Authoritativeness / Trustworthiness
Google が品質評価に使う4つの軸。「経験・専門性・権威性・信頼性」。2022年に Expertise → Experience が追加された。一次情報・実体験の記述や著者情報の明記が評価に直結する。
SEO
Core Web Vitals
Core Web Vitals（コアウェブバイタル）
Googleが定めるページ体験の指標。LCP（最大コンテンツの描画）・INP（インタラクションから次の描画）・CLS（視覚的安定性）の3指標でサイトのUXを評価する。ランキング要因のひとつ。
SEO
Search Intent（検索意図）
Search Intent
ユーザーが検索するときの目的。情報収集型（Know）・ナビゲーション型（Go）・購買検討型（Do）・取引型（Buy）の4種類に分類される。コンテンツ設計の前にキーワードの意図を正確に把握することが上位表示の前提。
SEO
Featured Snippet
Featured Snippet（強調スニペット）
Google の検索結果ページ最上部に表示される回答ボックス。「Position 0」とも呼ばれる。文章・リスト・表の形式があり、質問形式のキーワードで獲得しやすい。LLMO 対策とも密接に関連する。
SEO
Bounce Rate（直帰率）
Bounce Rate
サイトに訪問後、他のページへ移動せずに離脱したセッションの割合。GA4 では定義が変わり「エンゲージメントのなかったセッション」として計測される。高いほどコンテンツとユーザーの期待にギャップがある可能性を示す。
SEO
Backlink（被リンク）
Backlink / Inbound Link
外部サイトから自サイトへのリンク。Googleの権威性評価に影響する重要因子。質の高いドメインからのリンクほど効果が大きい。スパムリンクは Disavow ツールで否認できる。
SEO
SERP
Search Engine Results Page（検索結果ページ）
検索エンジンが表示する結果ページ全体のこと。オーガニック結果・広告・Featured Snippet・ナレッジパネル・画像・動画など複数の形式が混在する。SERP の構成を把握してからコンテンツ形式を決めることが重要。
SEO
Keyword Difficulty（KD）
Keyword Difficulty
あるキーワードで上位表示することの難易度を 0〜100 でスコア化した指標（Semrush・Ahrefs など）。競合サイトのドメイン強度や被リンク数をもとに算出される。新規サイトは KD の低いロングテールから攻めるのが基本。
SEOツール
Canonical（カノニカル）
Canonical Tag
重複コンテンツを解消するための HTML タグ（<link rel="canonical">）。複数 URL で同じ内容が公開されている場合に、正規 URL を Google に伝える。ECサイトのパラメータ URL などで多用される。
SEO
Domain Rating / Domain Authority
DR（Ahrefs）/ DA（Moz）
ドメイン全体の被リンクの質・量をもとにサードパーティツールが算出する権威性スコア。Google の公式指標ではないが、競合分析や被リンク獲得の優先順位付けに広く活用される。
SEOツール
LLMO
Large Language Model Optimization
ChatGPT・Claude・Gemini などの LLM（大規模言語モデル）が回答に引用・参照しやすいようコンテンツを最適化すること。E-E-A-T の強化・構造化された記述・明確な定義文の作成などが有効とされる。SEO の進化形として注目されている。
LLMO・AIO
AIO
AI Overview（旧 SGE）/ AI Optimization
Googleの検索結果上部に表示されるAI生成の回答サマリー（AI Overview）、またはAIに最適化されたコンテンツ戦略（AI Optimization）を指す。AI Overview に引用されることで、クリックなしに認知される「ゼロクリック時代」への対応が必要。
LLMO・AIO
GEO
Generative Engine Optimization
AI 検索エンジン（Perplexity・ChatGPT Search・Gemini など）での引用・回答採用を最適化する手法。LLMO とほぼ同義で使われることが多い。統計・一次情報・明快な定義文を含むコンテンツが引用されやすい傾向がある。
LLMO・AIO
RAG
Retrieval-Augmented Generation（検索拡張生成）
LLM が回答生成時に外部データベースから関連情報を取得して使用する仕組み。学習データ以降の最新情報や社内ドキュメントを参照できるようになる。AI Overview や ChatGPT Search もこの仕組みを使ってWebページを参照している。
LLMO・AIO
Prompt Engineering
Prompt Engineering（プロンプトエンジニアリング）
AI に対して意図した出力を引き出すための指示文（プロンプト）を設計するスキル。役割指定・出力形式の指定・具体例の提示（Few-shot）などのテクニックがある。ライター業務においては記事構成・リサーチ・推敲の効率化に直結する。
LLMO・AIO
Hallucination（ハルシネーション）
Hallucination
LLM が事実と異なる情報をあたかも正確であるかのように生成してしまう現象。ライターとして AI を活用する際は必ず一次情報でファクトチェックすることが必須。RAG の導入や温度パラメータの調整で軽減できる場合がある。
LLMO・AIO
Context Window
Context Window（コンテキストウィンドウ）
LLM が一度に処理できるテキストの最大量（トークン数で表される）。会話履歴・添付ドキュメントを含む全入力がこの上限に収まる必要がある。Claude の場合は最大 200K トークン。長文ドキュメントの分析や書籍要約に影響する。
LLMO・AIO
CAC
Customer Acquisition Cost（顧客獲得コスト）
新規顧客1人を獲得するためにかかった平均コスト。マーケティング費用 ÷ 新規顧客数で算出。LTV とセットで見て、LTV > CAC であれば事業として健全と判断できる。
セールスライティング
Churn Rate（解約率）
Churn Rate
一定期間内に既存顧客・ユーザーが離脱した割合。SaaS ビジネスでは月次 Churn Rate が重要指標。低いほど顧客維持ができている証拠。
セールスライティング
Conversion Rate（CVR）
Conversion Rate（コンバージョン率）
訪問者のうち、目標アクション（購入・登録・問い合わせなど）を実行した割合。コンバージョン数 ÷ 訪問者数 × 100 で算出。LP 改善の主要指標。
コピーライティング
CTR
Click-Through Rate（クリック率）
広告や検索結果が表示された回数のうち、クリックされた割合。クリック数 ÷ インプレッション数 × 100。広告クリエイティブや検索タイトルの効果を測る指標。
コピーライティング
LTV
Lifetime Value（顧客生涯価値）
1人の顧客が取引期間中にもたらす総収益の予測値。LTV = 平均購入単価 × 購入頻度 × 顧客継続期間。CAC と比較して施策の収益性を評価する。
セールスライティング
MQL / SQL
Marketing Qualified Lead / Sales Qualified Lead
MQL はマーケティング部門が「営業に渡せる」と判断したリード。SQL は営業が商談化できると判断したリード。BANT（予算・権限・ニーズ・タイミング）で MQL から SQL に昇格させる。
セールスライティング
ROAS
Return on Ad Spend（広告費用対効果）
広告費1円あたりの売上を示す指標。売上 ÷ 広告費 × 100（%）で算出。ROI と異なり原価等を含まない純粋な売上ベースの効率指標。
コピーライティング
TOFU / MOFU / BOFU
Top / Middle / Bottom of the Funnel
マーケティングファネルの3段階。TOFU（認知）は幅広い層への教育コンテンツ、MOFU（検討）は比較・事例コンテンツ、BOFU（購買）はデモ・価格ページなど。段階に応じてコンテンツ形式と CTA を変える。
コンテンツライティング
Buyer Persona（バイヤーペルソナ）
Buyer Persona
理想的な顧客像を半架空の人物として詳細に定義したもの。年齢・役職・課題・情報収集行動などを含む。コンテンツのトーン・テーマ・チャネル選定の基準になる。
コンテンツライティング
CPC
Cost Per Click（クリック単価）
広告1クリックあたりのコスト。Google 広告・Meta 広告など PPC（クリック課金型）広告の主要指標。競合が多いキーワードほど CPC が上昇する傾向がある。
コピーライティング
USP
Unique Selling Proposition（独自の強み）
競合と差別化できる自社独自の価値・提案。コピーライティングや LP 制作において「なぜこの商品・サービスを選ぶのか」を明確にするための基軸になる。
コピーライティング
CTA
Call to Action（行動喚起）
読者に次のアクションを促す文言・ボタン・デザイン要素。「無料で始める」「資料をダウンロード」など動詞＋ベネフィットの形が効果的。配置・色・文言の A/B テストが CV 改善に直結する。
ライティング
Above the Fold
Above the Fold（ファーストビュー）
スクロールせずに見えるページの最初の表示領域。LP や記事ではこのエリアにキャッチコピー・ベネフィット・CTA を置くことが鉄則。モバイルとデスクトップで見える範囲が異なる点に注意。
ライティング
Headline Formula（見出しの型）
Headline Formula
クリック率・読了率を高める見出しの定型パターン。代表例：数字（「5つの方法」）・How To（「〜する方法」）・Question（「〜していませんか？」）・List（「〜完全ガイド」）。英語圏でも同じ型が効果的に機能する。
ライティング
PREP法
PREP（Point / Reason / Example / Point）
文章の論理構造のフレームワーク。結論→理由→具体例→結論の順で書くことで、読み手に伝わりやすい文章になる。Web 記事・メール・SNS 投稿など幅広い場面で活用できる。
ライティング
Reading Level（読みやすさ）
Reading Level / Readability
文章の読みやすさを示す指標。英語では Flesch Reading Ease スコアが使われる。SurferSEO でも計測可能。専門用語の多用・一文の長さ・段落の構成が読みやすさに影響する。ターゲット読者に合わせたレベルで書くことが重要。
ライティングツール
Content Score
Content Score（SurferSEO）
SurferSEO が算出するコンテンツの SEO 品質スコア（0〜100）。上位表示ページとの比較をもとに、キーワード密度・見出し数・語数・画像数などを総合評価する。目安として 67 以上を目指すと良いとされる。
ツールSEO
Organic Traffic（オーガニックトラフィック）
Organic Traffic
検索エンジンの自然検索結果（広告を除く）経由で訪れたトラフィック。Google サーチコンソールや Semrush で計測できる。SEO 施策の成果を測る最重要指標のひとつ。
ツールSEO
Impression（インプレッション）
Impression（表示回数）
コンテンツや広告が画面に表示された回数。Google サーチコンソールでは「検索結果に表示された回数」として計測される。CTR と掛け合わせてクリック数を予測する際に使う。
ツール
